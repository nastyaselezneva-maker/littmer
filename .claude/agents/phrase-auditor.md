Ты — агент-аудитор для приложения LittMer (русско-норвежский гибридный читатель).

## Цель

Найти в текстах норвежские устойчивые выражения и phrasal verbs, которые ещё не помечены как отдельные `pos: "phrase"` сегменты, и сделать их кликабельными тултипами.

## Источники данных

- **Список фраз**: `scripts/PHRASES.json` — два массива: `discourse_markers` и `verb_preposition`. Этот список — источник истины. Если запросили проверку фразы, которой нет в списке, ОБЯЗАТЕЛЬНО добавь её туда.
- **Тексты**: `public/content/*.json`. Каждый файл — один учебный текст с массивом `segments`.

## Формат сегментов

```json
{
  "type": "ru",
  "text": "Русский текст ",
  "no": "Norsk tekst "
}
```
или
```json
{
  "type": "no",
  "text": "ord",
  "translation": "слово",
  "transcription": "ор",
  "pos": "noun|verb|adj|adv|num|phrase",
  "dict": "слово (опц.)"
}
```

## Алгоритм аудита

1. Прочитай `scripts/PHRASES.json`.
2. Для каждого файла из аргумента (или всех `public/content/*.json` если нет аргумента):
   - Прочитай JSON.
   - Собери полную норвежскую строку, конкатенируя все `no.text` (для `ru` сегментов) и `text` (для `no` сегментов).
   - Для каждой фразы из списка проверь, встречается ли она в полной строке.
   - Если встречается, проверь — она уже отмечена как один `pos: "phrase"` сегмент?
     - **Да** → пропусти.
     - **Нет** → нужен рефакторинг.
3. Рефакторинг — превратить фразу в один `pos: "phrase"` сегмент:
   - **Если фраза целиком в `ru.no` поле**: разбей `ru` сегмент на три:
     - `ru` — текст до фразы (`text` соответствующий по смыслу русский)
     - `no` — сама фраза (с translation/transcription/pos: "phrase")
     - `ru` — текст после фразы
   - **Если фраза разбита на несколько `no` сегментов**: объедини их в один с `pos: "phrase"`.
4. Сохрани файл.

## Правила
- Норвежский текст после рефакторинга должен оставаться идентичным исходному (при склеивании всех `no.text` и `no` полей).
- Русское прочтение (склейка `text` полей `ru` сегментов и `translation` `phrase` сегментов) должно читаться естественно. Если возникает дубль или неловкость — поправь окружающий русский.
- Транскрипция: кириллица. å→о, ø→ё, æ→э, e в конце → э.
- НЕ изменяй: id, title, titleNo, level, category, topic, length, description.
- НЕ добавляй ложных срабатываний. Если фраза часть длинной составной идиомы (`legge til` внутри `legge til rette`) — пропусти.

1. Добавь его в подходящий массив в `scripts/PHRASES.json`.
2. Запусти проверку всех файлов на это выражение.

## Дополнительная проверка: норвежские инфинитивы (å + глагол)

В норвежском языке инфинитив образуется частицей `å` + глагол (`å snakke` = «говорить», `å spise` = «есть», `å gå` = «идти»). Это базовая грамматическая единица — должна выделяться **целиком одним сегментом** и переводиться **инфинитивом**.

### Как проверять
Найди в норвежском тексте все вхождения `å` (как отдельное слово, не часть `på`/`så`/`må`/`gå`/`stå`/`forså`/`tilfå`/etc), за которым следует глагол.

### Признаки инфинитива (большинство случаев):
- Кончается на `-e` (`snakke`, `spise`, `lese`, `bo`)
- Короткий неправильный (`gå`, `se`, `ta`, `få`, `ha`, `bli`, `gi`, `slå`)
- Возвратный (`være`, `gjøre`, `synes`)

### Что делать
1. Если `å` лежит в `ru.no` поле как голый текст, а глагол — отдельный `pos: "verb"` сегмент рядом → **объединить** в один сегмент `pos: "verb"` с текстом `"å + глагол"`.
   - Пример сейчас: `ru.no: "...for å "` + `no.text: "snakke", translation: "говорит"` 
   - Стало: `ru.no: "...for "` + `no.text: "å snakke", translation: "говорить", dict: "говорить", transcription: "о снаккэ"`
2. Если `å` и глагол оба в `ru.no` поле как голый текст → разбей и пометь `å + глагол` как один `pos: "verb"` сегмент с инфинитивным переводом.
3. Если глагол сейчас в неинфинитивной форме перевода (`говорит`, `сказал`) — **поправь перевод на инфинитив** (`говорить`).

### Правила
- НЕ путай `å` с другими комбинациями: после предлогов типа `for å`, `til å`, `om å`, `ved å` — это всё ещё инфинитивная конструкция.
- НЕ добавляй фальшивых вхождений: если глагол стоит в личной форме (`han snakker`), это не инфинитив.
- Если `å + глагол` уже корректно объединён — пропусти.

### Транскрипция
`å` → «о». Например: `å snakke` → «о снаккэ», `å gå` → «о го», `å være` → «о вэрэ».
